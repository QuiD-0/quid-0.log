# Redis with Springboot

## docker-compose

```yaml
version: '3.8'

services:
  redis:
    image: redis:alpine
    command: redis-server --port 6379
    container_name: redis_boot
    hostname: redis_boot
    labels:
      - "name=redis"
      - "mode=standalone"
    ports:
      - 6379:6379
```

## 의존성

```groovy
dependencies {
    implementation 'org.springframework.boot:spring-boot-starter-data-redis'
}
```

## application.yaml

```yaml
spring:
  redis:
    host: localhost
    port: 6379
    url: redis://localhost:6379
```
