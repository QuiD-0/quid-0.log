나이가 가장 적은 사람을 찾고싶다  -> 풀스캔&#x20;

| 나이 | 인덱스 주소 | . | id | name | age |
| -- | ------ | - | -- | ---- | --- |
| 21 | 1      |   | 1  | AAA  | 21  |
| 28 | 4      |   | 2  | BBB  | 30  |
| 30 | 2      |   | 3  | CCC  | 39  |
| 39 | 3      |   | 4  | DDD  | 28  |

나이순으로 정렬되어있기 때문에 하나만 조회하면 된다.
